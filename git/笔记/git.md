#### SSh免密登錄

在c/user/asus/  目錄下：右键 git bash here  生成.ssh密钥目录：

​	ssh-keygen -t rsa -C  用户邮箱   

   -t 对应使用哪种加密方法来形成

​    rsa 非对称加密

   -C  描述

然后将生成的公钥给github或者gitee
